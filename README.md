# Portfolio Website

A modern, responsive portfolio website showcasing personal projects, skills, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Smooth scrolling between sections with active link highlighting
- **Modern UI**: Clean and professional design with gradient backgrounds and card layouts
- **Interactive Elements**: Hover effects, scroll animations, and mobile menu toggle

## Sections

- **Home**: Welcome section with introduction
- **About**: Brief description and background
- **Skills**: Technical skills organized by category
- **Projects**: Showcase of completed projects
- **Contact**: Contact information and links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Responsive design principles

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Customize the content in `index.html` with your personal information
4. Update colors and styles in `styles.css` as needed
5. Deploy to your preferred hosting platform

## Customization

- Update personal information in `index.html`
- Modify color scheme in `styles.css` (see CSS variables in `:root`)
- Add or remove sections as needed
- Update project links and descriptions
